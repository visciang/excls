    Elixir Command Line Shell - a simple framework for writing line-oriented command interpreters.

    usage

     Usage

    Escript:

    
def project do
  [
    app: :mycli,
    deps: [
      {:excls, github: "visciang/excls", tag: "xxx"}
    ],
    escript: [
      emu_args: "-noinput"
    ]
  ]
end

    Elixir script:

    #!/usr/bin/env -S elixir --erl -noinput

Mix.install([
  {:excls, github: "visciang/excls", tag: "xxx"}
])
